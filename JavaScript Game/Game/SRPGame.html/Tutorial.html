TUTORIAL

  1. First, you need to enter a player name. The program will automatically pick an opponent for you.
  2. Then, pick either "SCISSOR" , "ROCK" OR "PAPER".
  3. The computer AI would randommly pick a choice to play with you.
  4. The program would automatically generate the result for each round and end the game after round 5 and calculate the final score.
  5. The player can click the "End Game" button to see the final result and link to home page.
  6. Besides, the player can click the "Reset" button to reset the score and start again.
